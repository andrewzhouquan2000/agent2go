## 1. 产品概述

### 1.1 产品定位

**Agent2Go** = AI Agent 工作流编排平台

```
┌─────────────────────────────────────────────────┐
│              Agent2Go 平台                       │
├─────────────────────────────────────────────────┤
│  🤖 创建 AI Agent    🔗 编排工作流              │
│  ⚡ 执行任务         🔌 集成第三方              │
│  👥 团队协作         📊 数据分析                │
└─────────────────────────────────────────────────┘
```

### 1.2 核心价值

```
用户痛点                    Agent2Go 解决方案
─────────────────────────────────────────────────
人工操作效率低      →      AI Agent 自动化 (10 倍效率)
多系统切换繁琐      →      一站式集成平台
技术门槛高          →      可视化编排，无需编程
数据孤岛            →      统一数据流管理
```

```
┌─────────────────────────────────────────────────────────┐
│  users (用户表)                                         │
├─────────────────────────────────────────────────────────┤
│  id (UUID, PK)      │  主键                             │
│  email (VARCHAR)    │  邮箱 (唯一)                       │
│  name (VARCHAR)     │  昵称                             │
│  avatar (VARCHAR)   │  头像 URL                          │
│  created_at (TS)    │  创建时间                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  agents (Agent 表)                                      │
├─────────────────────────────────────────────────────────┤
│  id (UUID, PK)      │  主键                             │
│  user_id (UUID, FK) │  所属用户 → users.id              │
│  name (VARCHAR)     │  Agent 名称                        │
│  model (VARCHAR)    │  AI 模型 (GPT-4o 等)                │
│  config (JSONB)     │  配置 (Tools/Memory)              │
│  prompt (TEXT)      │  System Prompt                    │
│  status (VARCHAR)   │  状态 (idle/running/error)        │
│  created_at (TS)    │  创建时间                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  workflows (工作流表)                                   │
├─────────────────────────────────────────────────────────┤
│  id (UUID, PK)      │  主键                             │
│  user_id (UUID, FK) │  所属用户 → users.id              │
│  name (VARCHAR)     │  工作流名称                        │
│  nodes (JSONB)      │  节点定义                         │
│  edges (JSONB)      │  连接关系                         │
│  status (VARCHAR)   │  状态 (draft/active)              │
│  created_at (TS)    │  创建时间                         │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  tasks (任务表)                                         │
├─────────────────────────────────────────────────────────┤
│  id (UUID, PK)      │  主键                             │
│  agent_id (UUID, FK)│  执行 Agent → agents.id           │
│  workflow_id (UUID) │  所属工作流 → workflows.id        │
│  status (VARCHAR)   │  状态 (pending/running/done/fail) │
│  input (JSONB)      │  输入数据                         │
│  output (JSONB)     │  输出数据                         │
│  logs (TEXT)        │  执行日志                         │
│  error (TEXT)       │  错误信息                         │
│  created_at (TS)    │  创建时间                         │
└─────────────────────────────────────────────────────────┘
```
